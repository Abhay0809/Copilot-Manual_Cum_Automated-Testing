# Checkout Test Scenarios

1. Verify user can proceed to checkout from cart with valid items.
2. Verify guest/user authentication behavior during checkout is as expected.
3. Verify delivery and billing address details are displayed correctly.
4. Verify order summary matches cart contents and totals.
5. Verify placing order with valid details succeeds.
6. Verify error handling when required checkout fields are missing.
7. Verify payment form validates invalid card number format (if applicable).
8. Verify payment form validates expired card date (if applicable).
9. Verify payment form validates CVV length/format (if applicable).
10. Verify success confirmation is shown after order placement.
11. Verify duplicate submit prevention when place-order is clicked multiple times.
12. Verify checkout flow handles browser refresh/back navigation safely.
