# Signup Test Scenarios

1. Verify user can register with all valid mandatory fields.
2. Verify signup fails when mandatory fields are left blank.
3. Verify email format validation for invalid email pattern.
4. Verify signup fails for an already registered email address.
5. Verify password field enforces minimum required length.
6. Verify password and confirm password mismatch validation (if available).
7. Verify special characters are accepted or rejected as per field rules.
8. Verify name field handles maximum character length correctly.
9. Verify successful signup redirects user to expected post-registration page.
10. Verify confirmation message/account created status is displayed.
11. Verify user session starts automatically after successful signup (if expected).
12. Verify form data is not lost unexpectedly when one field fails validation.
